QWOP Reinforcement Learning Agent

A PPO-trained agent to play the physics-based QWOP game

Project Overview

This project implements a reinforcement learning model using the Proximal Policy Optimization (PPO) algorithm from the Stable Baselines3 library. The model is trained to play the QWOP game, a challenging physics-based browser game where players must control a runner's legs using only four keys. Through careful environment setup and training using custom Gym wrappers, the agent learns to navigate the track effectively.
