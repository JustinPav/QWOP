QWOP Reinforcement Learning

Training an agent to play QWOP using PPO and Stable-Baselines3

View on GitHub

Overview

This project implements a reinforcement learning agent to play the challenging physics-based game QWOP using the PPO algorithm from Stable Baselines3.

Setup Instructions (Ubuntu 22.04)

  1. Install Python 3.10+ and pip.
  2. Install Google Chrome and ChromeDriver (version-matched).
  3. Create and activate a Python virtual environment.
  4. Install required packages: 
    pip install qwop-gym jinja2 typeguard stable-baselines3[extra] gymnasium tensorboard
  5. Bootstrap and patch the QWOP environment:
    6. qwop-gym bootstrap
curl -sL https://www.foddy.net/QWOP.min.js | qwop-gym patch

Usage

To train the model:

python ppo_training.py

To test the trained model:

python ppo_test.py
